About The DPP-AVR Example
=========================
This directory contains the DPP example for the 8-bit AVRmega processor family
with the IAR-AVR compiler.

***
NOTE:
The Spy build configuration exceeds the 8KB code size limit of the free
IAR-AVR KickStart version and therefore it fails linking. Please use the
unlimited IAR_AVR toolset for QS. 
***
